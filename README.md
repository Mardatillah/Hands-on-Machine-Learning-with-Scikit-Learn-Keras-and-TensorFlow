# Hands on Machine Learning with Scikit Learn Keras and TensorFlow
Chapter 1 — The Machine Learning Landscape

- Introduction to what machine learning is, its applications, and challenges.

Chapter 2 — End-to-End Machine Learning Project

- Overview of the steps involved in a machine learning project, from data collection to model deployment.

Chapter 3 — Classification

- Detailed discussion on classification problems, algorithms like k-Nearest Neighbors, and evaluation metrics.

Chapter 4 — Training Models

- Techniques such as linear regression, gradient descent, and regularization in training machine learning models.

Chapter 5 — Support Vector Machines

- Introduction to SVMs for classification and regression tasks, including kernel methods.

Chapter 6 — Decision Trees

- Explanation of decision tree algorithms, including training and visualizing decision trees.

Chapter 7 — Ensemble Learning and Random Forest

- Techniques like bagging, boosting, and random forests for improving model accuracy.

Chapter 8 — Dimensionality Reduction

- Methods to reduce the number of features in the data, such as PCA and LLE.

Chapter 9 — Unsupervised Learning Techniques

- Covers clustering, anomaly detection, and density estimation.

Chapter 10 — Introduction to Artificial Neural Networks with Keras

- Basic neural network concepts and implementation using Keras.

Chapter 11 — Training Deep Neural Networks

- Techniques for optimizing deep neural networks, including avoiding vanishing/exploding gradients and using advanced optimizers.
